# 我的 HTML 小程式集合

這個倉庫包含一個總入口頁和多個小程式，每個小程式放在獨立的資料夾中。

## 使用方式
- 總入口頁網址：  
  `https://username.github.io/my-html-projects/`

- 小程式網址：  
  - `https://username.github.io/my-html-projects/project1/`  
  - `https://username.github.io/my-html-projects/project2/`  
  - `https://username.github.io/my-html-projects/project3/`

## 新增小程式
1. 建立新資料夾 (例如 `project4`)  
2. 在資料夾裡放 `index.html`  
3. 在總入口頁的清單加一行：  
   ```html
   <li><a href="project4/">小程式 4</a></li>
